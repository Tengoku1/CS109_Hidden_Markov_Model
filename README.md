# CS109_Hidden_Markov_Model
This is my CS109 Project made using Python and various imports from scikitlearn and pandas. This project is based on the theory of Hidden Markov Models and how they can be used to predict on longitudal data how a chronic kidney disease would transition throughout many years using data found from the MIMIC-IV clinic and their biometric data on chronic kidney disease. My model was able to accuractly assess whether a transtion would occur at a 80% accuracy rating. Additionally, I included a Monte Carlo model to reduce uncertainty based on my algorithm.
